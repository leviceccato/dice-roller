## Usage

### Requirements

- Node 14

### Development

Install dependencies

```shell
npm i
```

Run development server on [https://localhost:3000](https://localhost:3000)

```shell
npm run dev
```

Build plugin into `dist` folder

```shell
npm run generate
```
